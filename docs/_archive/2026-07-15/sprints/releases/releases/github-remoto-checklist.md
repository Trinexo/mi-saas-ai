# Checklist de garantía remota (cuenta/permisos)

## Cuenta y permisos
- [ ] Cuenta con rol Maintain o Admin en el repositorio
- [ ] Login activo en GitHub CLI (`gh auth status`)

## Reglas en main
- [ ] PR obligatorio para merge
- [ ] 1+ aprobaciones requeridas
- [ ] `test-backend` requerido
- [ ] `build-frontend` requerido
- [ ] Rama al día antes de merge
- [ ] Admins también sujetos a reglas
- [ ] Force-push bloqueado
- [ ] Eliminación de rama principal bloqueada

## Evidencia mínima de cierre
- [ ] URL del PR de cierre
- [ ] Captura/checks en verde
- [ ] Aprobaciones registradas
- [ ] SHA mergeado en main
- [ ] Release note enlazado en docs/README.md

## Estado final
- [ ] Sprint marcado como cerrado en tablero
- [ ] Preparado inicio de Sprint 4
