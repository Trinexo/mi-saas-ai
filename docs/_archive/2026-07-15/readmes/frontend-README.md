# Frontend Sprint 1

## Requisitos

- Node.js 20+

## Configuración

1. Copiar `.env.example` a `.env`.
2. Instalar dependencias:

```bash
cd frontend
npm install
```

3. Arrancar app:

```bash
npm run dev
```

URL local: `http://localhost:5173`

## Flujo implementado

- Login/registro
- Selección oposición/materia/tema
- Generación de test
- Resolución y envío
- Pantalla de resultados
- Pantalla de progreso
- Panel admin de preguntas (listado + alta)
