# Sistema de gestión de contenido masivo

## Objetivo

Gestionar cientos de miles de preguntas sin perder control de calidad.

---

# Roles

autor

revisor

administrador

---

# Flujo de publicación

autor crea pregunta

↓

revisor valida

↓

publicación

---

# Versionado

Cada pregunta tiene:

version

fecha_actualizacion

autor

---

# Moderación

Usuarios pueden reportar:

error

ambigüedad

contenido desactualizado

---

# Herramientas de administración

búsqueda avanzada

filtros por tema

edición masiva

importación CSV
