# Arquitectura de aplicación móvil

## Objetivo

Ofrecer experiencia de estudio desde móvil.

---

# Tecnologías

React Native  
Flutter  

---

# Funciones principales

- realizar test
- revisar preguntas
- estadísticas
- notificaciones

---

# Sincronización

La app debe sincronizar con backend mediante API.

---

# Modo offline

Permitir:

descargar test

resolver sin internet

sincronizar después

---

# Notificaciones

Enviar:

recordatorios de estudio  
objetivos semanales  
progreso alcanzado

---

# Experiencia móvil

Interfaz simple:

- pregunta grande
- opciones claras
- navegación rápida
