# Plan de lanzamiento de plataforma SaaS

## Objetivo

Lanzar una plataforma de preparación de oposiciones como servicio online.

---

# Fase 1 – MVP

Duración estimada: 3 meses

Funcionalidades:

- registro usuarios
- banco de preguntas
- test básico
- corrección automática

---

# Fase 2 – Plataforma completa

Duración estimada: 6 meses

Añadir:

- simulacros
- estadísticas
- panel administrador
- sistema de suscripción

---

# Fase 3 – Escalado

Añadir:

- app móvil
- ranking
- retos
- aprendizaje adaptativo

---

# Modelo de negocio

Suscripción mensual:

- plan básico
- plan premium

---

# Métricas clave

Usuarios activos  
Test realizados  
Preguntas respondidas  
Conversión a pago  

---

# Estrategia inicial

Lanzar con:

- 3 oposiciones
- 20.000 preguntas
- simulador de examen
