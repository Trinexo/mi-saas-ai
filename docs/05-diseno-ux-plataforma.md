# Diseño UX de la plataforma

## Flujo del usuario

Inicio
→ seleccionar oposición
→ seleccionar tipo de test
→ realizar test
→ ver resultados

---

# Pantallas principales

## Home

Mostrar:

- oposiciones disponibles
- progreso usuario

---

## Configuración de test

Opciones:

- número preguntas
- materias
- temporizador

---

## Pantalla de examen

Elementos:

- pregunta
- opciones
- temporizador
- navegación

---

## Corrección

Mostrar:

- respuesta correcta
- explicación
- referencia normativa

---

## Estadísticas

Gráficos:

- progreso
- aciertos por tema
- tiempo medio

---

# Principios UX

- interfaz simple
- evitar distracciones
- navegación rápida
- lectura clara

---

# Diseño responsive

Debe funcionar en:

- móvil

- ordenador
